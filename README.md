# Tic tac toe websockets

Two players tic-tac-joe implemented with an HTML canvas and websockets 

![](screenshot.png)

## How to run

```
// compilation of .mts files in watch mode
$ tsc -w

// serve static files (localhost:8000)
$ python serve.py

// run server
$ node server.mjs
```

Go to http://localhost:8000
